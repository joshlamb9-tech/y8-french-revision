How to Use This Section

Don't just read! Research shows that re-reading is one of the weakest revision methods. Instead:

  1. Study one topic for 25 minutes using the activities on each page
  2. Test yourself using the Brain Dump or Flashcard activities
  3. Return in 2-3 days to strengthen your memory (spaced practice)
  4. Mix it up - practice different tenses together so you learn when to use each one
